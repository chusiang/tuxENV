- Build Date: 2013/02/07
- Version: 1.1

If you like Vim, you can see my [vimrc](https://github.com/chusiang/vimrc).

## Setup

Clone tuxENV.
	
	git clone git@github.com:chusiang/tuxENV.git
	cd tuxENV/

Install

	make install

Update

	make update
	
Backup

	make backup
